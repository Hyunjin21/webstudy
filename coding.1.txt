[HTML 기본 문법]
TAG(태그) - 열리는 태그와 닫히는 태그 
ex) <strong> creating <u> web </u> pages </strong>
<strong>'중요한 것을 진하게 표시해야 한다'를 설명하는 태그
<u>underline 밑줄 태그
즉, 태그는 설명을 나타낸다.

쉽다 = 중요하다 / 가장 자주 사용하는 부품들 = 처음 기본으로 배우는 것

<h1>~<h6> 제목을 나타내는 태그 - 숫자가 올라갈수록 글씨가 작아짐 
h숫자 - 일반적인 텍스트보다 글씨가 두껍다. 줄바꿈이 된다.
ex) <h1>W3C</h1>

150+ 태그 중 26개 정도

줄바꿈 태그 <br> vs <p>

<br> 줄바꿈
- 쓰는 만큼 여러번 줄바꿈 가능/각각 한줄씩 줄바꿈
- 닫히는 태그를 쓰지 않음
- 원하는 만큼 간격을 줄 수 있음 (장점)

<p>~</p> 단락표현
- 웹페이지를 정보로서 가치있게 해줌
- 단락과 단락의 간격이 정해진 여백만큼 고정되어 시각적인 자유도가 떨어진다.

+) css 문법 - 태그와 태그 사이 여백을 나타내는 코드
- <p> 태그 정교하게 제어가능
ex) <p style = "margin-top:40px;">~
